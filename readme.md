# Interactive Refrigerator Magnet To Reduce Food Waste

<p>
This is an iOS application which was used as a prototype to faccilitate my honours year dissertation project. The project involved researching persuasive design to implement and evaluate an interactive refrigerator magnet with the goal of reducing household food waste. The final paper can be viewed <a href="https://www.joekadi.com/assets/honours_diss.pdf" target="_blank">here</a>. The project received a grade A.
</p>
